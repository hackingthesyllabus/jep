📄 Quick Import from Template File
Upload a .txt or .docx file written in the Jeopardy template format to instantly fill this entire board — categories, clues, answers, and Daily Doubles included. Much faster than entering clues one by one.
TEMPLATE FORMAT
GAME: My Trivia Night — sets the game title
BASE: 200 — base dollar value (produces $200, $400, $600…)
CATEGORY: Science — starts a new category column
Q: The clue text shown to players — a clue
A: What is the correct response? — the answer
DD Q: Clue text — marks that clue as a Daily Double
FINAL JEOPARDY — begins the Final Jeopardy section
# Lines beginning with # are ignored (use for comments)
# Blank lines are ignored

Rules: Every Q must be followed immediately by an A. All categories should have the same number of Q/A pairs (3–5). Max 6 categories. Works with both .txt and .docx files.
